# qml-template

This project can be used to create a minimal qml project for the [qml course](https://nicolasgourdiersurgar.github.io/learnqml/)

## How to build

```bash
mkdir build
cd build
cmake .. -DQt6_DIR=/path/to/qt/6.8.6/gcc_64/lib/cmake/Qt6
cmake --build .
```

## How to run

```bash
./Linux-x86_64/qmlApp
```
